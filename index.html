🔐 SPAR POC

Passkey · OIDC Authorization Code Flow · Scoped Claims

🔑 Passkey & Login

Register a FIDO2/WebAuthn passkey tied to the currently logged-in account, or authenticate using standard email & password. Logout clears the CDC session cookie.

👤 Session & Account

Verify whether a CDC session is active and retrieve the full account payload (profile, data, preferences). Full output is sent to the browser console — open DevTools → Console to inspect the JSON.

🪙 OIDC Proxy Flow

When Postman triggers the authorization code flow, CDC redirects to this page with ?context=... in the URL. The page automatically detects this, decodes the context, retrieves the active session and redirects back to CDC — which issues an authorization code to Postman's callback URL (https://oauth.pstmn.io/v1/callback). Watch the status bar above to see each step.
